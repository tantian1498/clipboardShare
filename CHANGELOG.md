# Changelog

## [1.2.2] - 2026-05-01

### 新增
- 历史记录批量删除：多选、全选、Shift 范围选择
- 删除操作同步到远端设备（基于 deleteVersion 版本号轮询）

### 修复
- 设置图标异常
- 时间被按钮覆盖
- 历史记录持久化到服务端

## [1.2.0] - 2026-04-29

### 新增
- 图片剪贴板同步
- UI 重构（卡片式历史列表、搜索、筛选标签、设置面板）
- 应用内更新检查与下载
- 历史记录持久化（服务端 history.json）

### 新增

### 新增
